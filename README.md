# serialPortTool

## 概述

基于jSerialComm的二次封装

## 依赖

- [jSerialComm](https://github.com/Fazecast/jSerialComm)

引入依赖

```xml
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
	
	<dependency>
	    <groupId>com.github.yiaoBang</groupId>
	    <artifactId>serialPortTool</artifactId>
	    <version>2.11.0-alpha2</version>
	</dependency>
```
